#  Mini E-Commerce Website

A fully functional **mini e-commerce web application** built using **HTML, CSS, JavaScript, and Bootstrap 5**.  
This project focuses on real-world shopping functionality, not just UI design.

---

##  Features

-  Dynamic product listing using JavaScript
-  Live product search (by title, description, and category)
-  Add to cart functionality
-  Increase / decrease product quantity
-  Remove items from cart
-  Cart data stored using `localStorage`
-  SweetAlert2 for user-friendly alerts
-  Fully responsive design with Bootstrap 5

---

##  Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **Bootstrap 5**
- **Font Awesome**
- **SweetAlert2**

---

##  What I Learned

- DOM manipulation using JavaScript
- Working with arrays and objects
- Managing application state with `localStorage`
- Building cart logic like real e-commerce systems
- Improving UX with alerts and responsive layouts
